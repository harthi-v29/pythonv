11.write a python program to accept a number and find the given number 
is palindrome or not?
Program:
n=int(input("Enter any num:"))

dup=n

sum=0

while n>0:

    rem=n%10

    sum=(sum*10)+rem

    n=n//10

if dup==sum:

 print("given number is palindrome")

else:

 print("given number is not palindrome")
